#streamelement 

---
#### A RNG based minigame that chat can play that aims to increase individual engagement.

- Create an RNG based stream element that chat can activate
	- Slots machine style game using emotes as the icons
	- Add a channel point reward that activates the element
	- Integrate chance based system using [streamer.bot](https://streamer.bot/)
		- IF "LOSER"
			- One or more emotes should be different
			- Activate a soundbite that signifies the player losing
		- IF "WINNER"
			- All three emotes show a Golden Ticket
			- Activate a soundbite that signifies the player winning
			- Showcase the winner on screen until a new winner is chosen
			- Grant VIP to the winner

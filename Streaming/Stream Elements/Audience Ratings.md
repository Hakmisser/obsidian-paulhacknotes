#streamelement 
---
#### A chat based rating system that aims to provide community engagement.

- Create a 5-star rating system based on chat input
	- Give a way for chat to rate the stream
		- Set up a chat match using [streamer.bot](https://streamer.bot/) and utilize emotes to lauch a command that influences the star rating
	- Display the star rating on screen for the viewers to see
	- If the star rating reaches 5-stars, activate an event on stream
		- Display a special overlay with visual and sfx
		- Activate a soundbite that matches the event
